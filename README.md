Mess Management System (Mess Regulator)
📌 Project Overview
Mess Management System is a web-based application developed for managing student mess activities efficiently.
It helps in tracking daily meals, managing expenses, calculating monthly costs, and maintaining transparent financial records.

This project was developed as part of our Web Engineering course.
