
# 🍽️ Mess Management System (Mess Regulator)

## 📌 Project Overview

**Mess Management System** is a web-based application developed to efficiently manage student mess activities.

It helps to:

* Track daily meals
* Manage expenses
* Calculate monthly costs
* Maintain transparent financial records

This project was developed as part of our **Web Engineering course**.
